# Proyecto Parqueadero

*Integrantes
Juan Antonio Bonett Sanchez

*Descripción del proyecto
Este programa simula un parqueadero C++.

El sistema puede:
- Mostrar el mapa del parqueadero
- Registrar la entrada de un vehículo
- Registrar la salida de un vehículo
- Calcular el valor a pagar
- Mostrar los espacios libres y ocupados

*Archivos
- `main.cpp´
- `run.cpp´
- `initialize_map.cpp`
- `display_map.cpp`
- `vehicle_entry.cpp`
- `vehicle_exit.cpp`
- `payment.cpp`

*Cómo compilar
Para compilar debes escribir lo siguiente en terminal; g++ "main.cpp" -o Parqueadero

*Cómo ejecutar
Para ejecutar deberas escribir lo siguiente en terminal (solo despues de haber compilado); ./Parqueadero
